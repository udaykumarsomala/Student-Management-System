# Student Management System

A full-stack CRUD application built using Django REST Framework, React.js, Bootstrap, and MySQL.

## Features

* Add Student
* View Students
* Update Student Details
* Delete Student
* Search Students
* Form Validation
* REST API Integration

## Technologies Used

### Backend

* Python
* Django
* Django REST Framework
* MySQL

### Frontend

* React.js
* Axios
* Bootstrap

## Project Structure

School Management System

├── SchoolProject

├── StudentApp

├── front-end

└── manage.py

## API Endpoints

GET /student

POST /student

PUT /student/{id}

DELETE /student/{id}
