# Student Management System

A full-stack CRUD application built using Django REST Framework, React.js, Bootstrap, and MySQL.

## Features

* Add Student
* View Students
* Update Student Details
* Delete Student
* Search Students
* Form Validation
* REST API Integration
* MySQL Database

## Technologies Used

### Backend

* Python 3.11
* Django 5
* Django REST Framework
* MySQL

### Frontend

* React.js
* Axios
* Bootstrap

## Project Structure

```text
School Management System

├── SchoolProject/

```

## API Endpoints

### Get Students

```http
GET /student
```

### Add Student

```http
POST /student
```

### Update Student

```http
PUT /student/{id}
```

### Delete Student

```http
DELETE /student/{id}
```
